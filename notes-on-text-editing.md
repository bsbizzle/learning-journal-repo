# Text Editors

### Standard Options
- Text Edit (comes with Mac)
- Notepad (comes with Windows)

### Third-Party Options 
- Notepad++ (Windows only)
- Text Wrangler (Mac only)

- BB Edit 
- Visual Studio Code
- Atom
- Brackets
- Sublime Text
